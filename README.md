# Fermionic quantum computation with Cooper's pair splitter: estates evolution by Clifford+T group and parity preserving

> [!NOTE]
> This enviroment is made with pixi. For more information visit [this](https://pixi.prefix.dev/latest/). 
> 
---

This project is an extension of [arXiv:2309.00447](https://arxiv.org/pdf/2309.00447).

Further analysis of the first device described in the study are made. [The first file](codes/TG_figures.py) includes the time dependent evolution due to a constant charged Hamiltonian for every gate of the universal set. [The second](codes/TG_figures2.py), shows the time dependent simulation with an rect pulse for every gate, given an initial state and the device Hamiltonian. [The last one](codes/parity_figures.py), is an proof of the universal set parity preserving in the Bloch sphare even and odd subspaces, for the device Hamiltonian.
